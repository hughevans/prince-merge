== Prince Merge

Generate multiple addressed PDFs with the wonderful PrinceXML.

http://princexml.com

== Usage

  ruby prince_merge.rb addresses.csv mailrun.pdf

The first argument is the CSV containing the recipients. The CSV file requires the following headers/columns:

  name,address_one,address_two,address_three,city,state,postal_code,country

The second argument is the file for output. Also there is an optional third argument for specifying a new HAML template:

  ruby prince_merge.rb addresses.csv mailrun.pdf letter.haml
